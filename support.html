Support

If you found an error, have trouble using the app, or need help with the study tool, contact us.

Email: long@formn400.org

We try to respond as soon as possible.

What to include in your message

To help us answer faster, please mention:

  • The question number or topic, if you spotted an error.
  • The language you were using when the issue appeared.
  • Your device and browser (or whether you were using the mobile app).
